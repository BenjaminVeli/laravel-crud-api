## Crud Laravel Api Students

### Initialize :

```sh
# Step 1: Install the dependencies defined in the composer.json file
composer install

# Step 2: Copy the example environment configuration file to .env
cp .env.example .env

# Step 3: Generate the application key, which is used for encrypted data and sessions
php artisan key:generate

# Step 4: Run the database migrations to create the necessary tables in the database
php artisan migrate

# Step 5: Start the development server to run the application locally
php artisan serve
```
